GeoPin

Find a user's location based upon their IP address. (MS SQL scripts/data and example code included)

Download .zip Download .tar.gz View on GitHub

So you wanted to do location-based web applications, but didn't want to be dependent upon HTML5 support, nor did you want to pay for subscription-based services for determining a user's location? No problem!

This project gives you everything you need to setup your MS SQL server-- and all of the data as well. Please see the attached code samples (available in both VB.Net and C#) for implementation. It's really quite simple and will save you a ton of time and money.

Coming soon: a add-on that allows for calculation of distance (as the bird flies) between the user's location and another (dynamic) location.
